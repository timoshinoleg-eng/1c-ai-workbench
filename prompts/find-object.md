# /find-object — Поиск использования объекта

**Когда использовать:** нужно найти все места в конфигурации, где используется определённый объект метаданных.

**Описание:** ищет упоминания объекта (справочника, документа, регистра) в коде других объектов.

**Шаги:**
1. Определить полное имя объекта (Метаданные.Справочники.Контрагенты)
2. Поискать через `grep-body` или `search-text` в `source-mirror/`
3. Собрать все вхождения с контекстом
4. Сгруппировать по типу использования: чтение, запись, вызов, форма

**Формат ответа:**
```
Объект: <полное имя>
Всего упоминаний: N

По типам:
- Модули объектов: N
- Формы: N
- Отчёты/обработки: N
- Запросы: N

Топ-5 мест для проверки:
1. <файл> — <контекст>
...
```
