# /review-bsl — Review BSL-кода

**Когда использовать:** нужно быстро найти проблемные места в BSL-модуле.

**Описание:** проверяет модуль по эвристикам: длинные процедуры (>100 строк), запросы в циклах, глубокая вложенность (>4 уровней), необработанные исключения, массовые side effects.

**Правила проверки:**
1. Процедуры > 100 строк — suspect (s)
2. Запрос (`Запрос.Текст =`) внутри цикла (`Пока`, `Для каждого`) — high (h)
3. Вложенность условий > 4 уровней — medium (m)
4. Обработка исключений без логирования — medium (m)
5. `Движения.` без проверки прав — medium (m)

**Формат ответа:**
```
Файл: <path>
Найдено: N замечаний

| # | Правило | Severity | Строка | Описание |
|---|---|---|---|---|
| 1 | Запрос в цикле | high | 42 | ... |
```

**Уверенность:** высокая для детерминированных правил, средняя для эвристик.
