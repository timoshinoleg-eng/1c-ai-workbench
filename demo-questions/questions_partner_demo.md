# Questions for a partner demo

Use this pack when you need to показать ценность стенда не разработчику, а партнёру или заказчику.

1. Где в нашей 1С считается сумма по документу или заказу?
2. Какие документы записывают продажи или движения по регистру?
3. Что придётся проверить, если бизнес попросит добавить новое поле в карточку товара?
4. Есть ли уже готовое место, связанное с телефонами клиентов или контрагентов?
5. Какой файл открыть руками, чтобы убедиться, что ответ не выдуман?

Expected behavior:

- answer is short and plain-language;
- contains one concrete file to open;
- names one or two exact places inside the file;
- says how a 1C specialist can verify it manually.
