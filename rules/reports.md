# Rules: Отчёты и обработки

**Scope:** файлы `Reports/*/Ext/*.bsl`, `DataProcessors/*/Ext/*.bsl`

**Принципы:**
- СКД — предпочтительный способ построения отчётов.
- Программные отчёты — только если логика не выражается через СКД.

**Проверки:**
- Используется СКД или программно-построенный отчёт?
- Если СКД — есть ли роль с правами на схемы?
- Нет ли жёстко зашитых параметров, которые должны быть настраиваемыми?
