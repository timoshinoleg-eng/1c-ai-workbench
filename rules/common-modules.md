# Rules: Общие модули

**Scope:** файлы `CommonModules/*/Ext/Module.bsl`

**Принципы:**
- Серверные общие модули — основной слой переиспользуемой логики.
- Клиентские — только для UI-логики.
- Вызов серверного из клиентского — строго через синхронные/асинхронные вызовы.

**Проверки:**
- Нет ли глобальных переменных состояния в модуле?
- Есть ли экспортные процедуры, которые не вызываются (dead code)?
- Соответствует ли нейминг назначению модуля (например, `РаботаСКурсамиВалют` — только про курсы)?
