# Rules: Регистры

**Scope:** файлы `Registers/*/Ext/*.bsl`

**Принципы:**
- Регистры — только хранилища данных. Логика — в документах.
- `RecordSet` — основной способ записи.

**Проверки:**
- Есть ли ресурсы? Для регистров накопления/бухгалтерии — обязательны.
- Есть ли индексы по измерениям, по которым чаще всего ищут?
- Измерения — не избыточны?
