# Rules: Запросы

**Scope:** файлы с текстом запросов и `Ext/*.bsl`

**Принципы:**
- Все `Запрос.Текст` должны быть читаемы (не конкатенация строк без форматирования).
- Использовать параметры, не конкатенацию строк.
- Не использовать `*` без явного перечисления полей в production.

**Проверки:**
- Есть ли динамические запросы через `++` / `+` строк? — risk.
- Есть ли `ВЫБРАТЬ *`? — medium.
- Есть ли `АвтоУпорядочивание`? — info.
- Есть ли `РАЗРЕШЕННЫЕ`? — проверка прав.
