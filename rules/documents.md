# Rules: Документы

**Scope:** файлы `Documents/*/Ext/*.bsl`

**Принципы:**
- `ObjectModule.bsl` — основная бизнес-логика документа.
- `ManagerModule.bsl` — утилиты и массовые операции.
- Проводка — строго в `ОбработкаПроведения()`.

**Проверки:**
- Есть ли `ОбработкаПроведения`? Для проводящихся документов — обязательна.
- Нет ли прямой записи в регистры из формы?
- Движения — только в модуле объекта, не в форме.
