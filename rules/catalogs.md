# Rules: Справочники

**Scope:** файлы `Catalogs/*/Ext/*.bsl`

**Принципы:**
- `ObjectModule.bsl` — проверки уникальности, реквизиты, обработка записи.
- `ManagerModule.bsl` — поиск, отбор, массовые операции.

**Проверки:**
- Есть ли проверка на дубли при записи (через `НайтиПоНаименованию` / `НайтиПоКоду`)?
- Есть ли обработка `ПередЗаписью` для валидации?
- Код — автоген или ручной?
