# Демо-образец для показа клиенту

## Простая идея

**1C AI Dev Workbench** — локальный AI-навигатор по вашей 1С-конфигурации.
Есть пример 1С. Мы не меняем его. Мы заранее построили по нему быстрый поиск.
Теперь можно спросить: «Где считается сумма?» — и получить место, которое можно открыть и проверить.

## Что показать

### Вопрос 1

**Где документ `ЗаказНаряд` считает сумму?**

Простой ответ:

> В документе `ЗаказНаряд` сумма считается в файле:
> `Documents/ЗаказНаряд/Forms/ФормаДокумента/Ext/Form/Module.bsl`.
>
> Там есть два важных места:
> - `обновитьСуммуПоТекущейСтрокеУслуг` — считает сумму строки: цена × количество.
> - `обновитьСуммуДокумента` — складывает все строки в общую сумму документа.

Как проверить руками:

1. Открыть файл выше.
2. Найти слова `Цена`, `Количество`, `СуммаДокумента`.
3. Убедиться, что сумма строки считается как `Цена * Количество`.

### Вопрос 2

**Где документ пишет продажи?**

Простой ответ:

> Продажи записывает документ `Реализация`.
> Нужный файл:
> `Documents/Реализация/Ext/ObjectModule.bsl`.
>
> В нём найдены места с `Движения.Продажи`:
> - `записатьДвиженияДокумента`
> - `выполнитьДвижениеПродажиОборот`

Как проверить руками:

1. Открыть файл выше.
2. Найти `Движения.Продажи`.
3. Посмотреть, какие поля туда записываются.

### Вопрос 3

**Что тронуть, если добавить новое поле в “Номенклатуру”?**

Простой ответ:

> Начинать надо с объекта `Номенклатура`.
> Основные места:
> - `Catalogs/Номенклатура.xml` — описание самого объекта.
> - `Catalogs/Номенклатура/Forms/ФормаЭлемента/Ext/Form.xml` — карточка элемента.
> - `Catalogs/Номенклатура/Forms/ФормаСписка/Ext/Form.xml` — список.
>
> После добавления поля надо проверить документы и отчёты, где используется номенклатура.

## Как показать, что это не презентация, а реальный поиск

Запустить:

```powershell
cd C:\1c-ai-workbench
.\scripts\06_healthcheck.ps1
```

Должны быть строки `[OK]`.

Потом выполнить пример поиска:

```powershell
.\tools\code-index-mcp\target\release\bsl-indexer.exe grep-body --path "C:\1c-ai-workbench\generated\index\source-mirror" --pattern "Движения.Продажи" --language bsl --limit 20
```

Он покажет файл и найденные процедуры.
